QUE ES HMTL

HTML, siglas en inglés de HyperText Markup Language (‘lenguaje de marcado de hipertexto’), hace referencia al lenguaje de marcado para la elaboración de páginas web. Es un estándar que sirve de referencia del software que conecta con la elaboración de páginas web en sus diferentes versiones, define una estructura básica y un código (denominado código HTML) para la definición de contenido de una página web, como texto, imágenes, videos, juegos, entre otros. Es un estándar a cargo del World Wide Web Consortium (W3C) o Consorcio WWW, organización dedicada a la estandarización de casi todas las tecnologías ligadas a la web, sobre todo en lo referente a su escritura e interpretación. HTML se considera el lenguaje web más importante siendo su invención crucial en la aparición, desarrollo y expansión de la World Wide Web (WWW). Es el estándar que se ha impuesto en la visualización de páginas web y es el que todos los navegadores actuales han adoptado

APRENDE MAS

A continuación se ofrece una lista de códigos HTML no estándar. Para especificar acciones determinadas, puede añadir atributos estándar a todos los códigos o puede aplicar atributos que son para códigos individuales. Pulse en cada código para obtener una descripción y otros atributos que se aplican a los códigos específicos: bgsound blink embed marquee nobr noembed wbr

bgsound: especifica un archivo de sonido para la página.

blink: especifica el texto que aparece de forma intermitente. Puede editar y visualizar este código en la vista Diseño; no obstante, no puede ver el texto intermitente en la página Diseño. Este código se indica mediante líneas de puntos alrededor del contenido.

embed: incluye un objeto de plug-in en el documento. Este código puede editarse y visualizare en la página Diseño.

marquee: define texto que se mueve en un documento. Este código puede editarse y visualizare en la página Diseño. Solo se ejecuta en Explorer 4.0 o posterior.

nobr: sobrescribe los saltos de línea en un bloque de texto. Este código puede editarse en la página Origen o en la vista Esquema, pero no puede visualizarse en la página Diseño.

wbr :Permite definir un salto de línea en un código -NOBR-. Este código se puede editar en la página Origen o en la vista Esquema, pero no se puede visualizar en la página Diseño.
